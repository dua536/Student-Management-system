# 🎓 Smart Student Management System

A comprehensive Student Management System built with Python and Streamlit, featuring advanced data structures and a modern GUI.

## 🚀 Features

- **Student Records Management** - Add, delete, and view student information
- **Advanced Data Structures** - BST, Stack, and Queue implementations
- **Batch Processing** - Queue-based student processing system
- **GPA Sorting** - Binary Search Tree for efficient GPA sorting
- **Undo/Redo Operations** - Complete action history system
- **Search & Analytics** - Advanced search and reporting capabilities
- **Export Functionality** - Export data to Excel and CSV formats
- **Modern GUI** - Professional dark-light theme with glassmorphism effects

## 🛠️ Technologies Used

- **Python 3.8+**
- **Streamlit** - Modern web application framework
- **Pandas** - Data manipulation and analysis
- **Matplotlib** - Data visualization and reporting
- **OpenPyXL** - Excel file operations
